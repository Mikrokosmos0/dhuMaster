# 自己先拟定好大论文的提纲

比如毕业论文的 Chapter 3 对应自己的某篇论文

宏观整体靠提纲，微观细节靠自己

请帮我完成以下任务

1. **翻译**：将源论文从英文翻译成学术中文
2. **重组**：按照硕士学位论文的章节结构重新组织内容
3. **扩充**：由于原论文篇幅限制，需要在以下方面进行扩充：
   - 增加背景介绍和研究动机的详细阐述
   - 补充方法的理论推导和实现细节
   - 扩展实验分析和讨论
   - 增加与上下文章节的衔接内容

# 一、翻译规范

## 1. 语言风格

使用规范的学术中文，避免口语化表达

- 使用"本文"、"本章"、"本节"等作为主语，或使用被动语态
- 句式简洁明了，逻辑清晰

## 2. 术语处理

- 专业术语首次出现时标注英文：中文术语(English Term（完整版）, 缩写)
- 全文术语翻译保持一致
- 常见术语参考表

## 3. 公式处理

- 保留原有公式的 LaTeX 代码
- 公式描述翻译成中文
- 确保公式编号连续
- 重要公式添加详细的符号说明

## 4. 引用处理

- 保留所有 `\cite{}` 命令
- 新增引用需要添加到主 bib 文件中
- 引用格式：文献[1]表明...或...如文献[1,2]所示

# 二、质量检查清单

完成后请检查以下项目:

## 内容检查

- 所有技术术语翻译一致
- 所有公式编号正确且有引用
- 所有图表有标题和引用
- 所有参考文献引用正确
- 内容逻辑清晰，层次分明
- 论文其他章节风格一致

## 格式检查

- 章节标题层次正确
- 图片路径正确
- 表格格式统一
- 公式格式规范
- 无编译错误

## 篇幅检查

- 达到目标篇幅要求
- 各部分比例合理
- 无明显冗余内容

# 三、常见问题处理

**Q1:** 原论文中的补充材料(Supplementary)如何处理？
**A:** 将重要内容整合到正文中，次要内容可放入论文附录。

**Q2:** 原论文与毕业论文其他章节有重复内容怎么办？
**A:** 在本章简要提及并引用其他章节，避免大段重复。如"关于XX的详细介绍见第X章"。

**Q3:** 如何保持与其他章节的一致性？

**A:** 参考已完成章节的写作风格、术语使用和格式规范
